# Jak pobrać PDF z oferty

Plik źródłowy oferty znajduje się tutaj:

- `docs/projectaanbod-website-nl.md`

Ponieważ repozytorium zawiera treść oferty w formacie Markdown, PDF można wygenerować w 2 prostych krokach:

## Opcja 1 (najprostsza, bez instalacji)

1. Otwórz plik `docs/projectaanbod-website-nl.md` w VS Code.
2. Włącz podgląd Markdown (`Ctrl+Shift+V`).
3. W podglądzie użyj **Print** / **Drukuj**.
4. Wybierz drukarkę: **Save as PDF / Zapisz jako PDF**.
5. Zapisz jako np. `projectaanbod-website-nl.pdf`.

## Opcja 2 (z poziomu GitHub/GitLab)

1. Otwórz plik `docs/projectaanbod-website-nl.md` w przeglądarce.
2. Użyj funkcji drukowania (`Ctrl+P`).
3. Wybierz **Zapisz jako PDF**.
4. Zapisz plik lokalnie.

## Co otrzymasz

Gotowy dokument PDF po niderlandzku z pełną strukturą oferty:
- klauzula poufności,
- zakres projektu,
- cena €400,
- opcjonalny hosting/utrzymanie €100/rok,
- bezpieczeństwo migracji i ciągłość działania,
- propozycje screenshotów i podpisów.
