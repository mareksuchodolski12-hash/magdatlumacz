## 8) Wspólne wytyczne SEO i skalowalności publikacji (wdrożenie CMS)

### H2: Standard techniczny SEO
- Jeden **H1** na podstronę, logiczna hierarchia **H2–H3**.
- Unikalny meta title i meta description dla każdej podstrony.
- Linkowanie wewnętrzne między: Usługi ↔ Języki ↔ Cennik ↔ Kontakt.
- Spójne słownictwo encji: tłumacz przysięgły, tłumaczenia urzędowe, tłumaczenia biznesowe, Holandia.

### H2: Skalowalna struktura treści
- Komponenty CMS: Hero, Sekcja usług, FAQ, CTA, E‑E‑A‑T, Tabela cennika.
- Reużywalne bloki treści dla landing pages językowych i branżowych.
- Możliwość rozbudowy o wersje językowe i strony lokalne bez zmiany architektury informacji.

### H2: Spójność E‑E‑A‑T globalnie
- Stały blok „O autorce / kwalifikacjach” na stronie głównej i w blogu.
- Stały blok „Poufność i standard współpracy” na stronach Usługi, Cennik, Kontakt.
- Aktualizacja treści eksperckich minimum raz na kwartał (sygnał świeżości i wiarygodności).
